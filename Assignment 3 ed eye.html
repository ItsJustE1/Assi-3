Honda Type R

Description

The Honda Type R is a high-performance version of the Honda Civic hatchback. It features a turbocharged 2.0-liter four-cylinder engine that produces 306 horsepower and 295 lb-ft of torque. The Type R also has a sport-tuned suspension, Brembo brakes, and a six-speed manual transmission.

Features

  • Turbocharged 2.0-liter four-cylinder engine
  • 306 horsepower and 295 lb-ft of torque
  • Sport-tuned suspension
  • Brembo brakes
  • Six-speed manual transmission
